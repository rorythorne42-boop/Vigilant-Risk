Vigilant Risk GitHub Pages upload

Files included:
- index.html
- robots.txt
- sitemap.xml
- CNAME

GitHub Pages steps:
1. Create a new public GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. In GitHub open Settings > Pages.
4. Set:
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
5. Save and wait for GitHub Pages to publish.

Custom domain:
This package already includes a CNAME file for:
www.vigilant-risk.com

DNS:
- Set www as a CNAME to <your-github-username>.github.io
- Point or forward the root domain to www.vigilant-risk.com

If you change the domain later, update CNAME, robots.txt, and sitemap.xml.
